Answer:

Chiz earned only -3/4 of his learning this month, it means he earned 1/4 less than his learning.

So, if we let x be his learning for this month, we can write the equation:

x - 1/4x = earnings for this month

Simplifying the equation, we get:

3/4x = earnings for this month

Now, we can substitute the given value of Chiz's earning last month and solve for x:

12,600 = 3/4x

x = 12,600 ÷ (3/4)

x = 16,800

Therefore, Chiz's learning for this month is P16,800.

Answer:

Chiz earned P9,450 this month.

Step-by-step explanation:

1. Start by assigning a variable to the unknown quantity we're trying to solve for. In this case, let's use variable E to represent Chiz's earnings this month.

2. Use the given information to write an equation that relates the variable E to the other quantities. We know that Chiz earned only 3/4 of his earnings last month, which was P12,600. So we can write:

E = 3/4 x P12,600

3. Simplify the equation using the rules of arithmetic. To multiply by 3/4, we can either multiply the numerator by 3 and then divide by 4, or we can multiply the whole fraction by 3/4. Either way, we get:

E = (3/4) x P12,600

E = P9,450

4. Check the solution to make sure it makes sense. In this case, we can check by verifying that if Chiz earned only 3/4 of P12,600, his earnings this month would indeed be P9,450. Therefore, our solution is correct.

So Chiz earned P9,450 this month.
