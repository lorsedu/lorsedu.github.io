= [x - 3 + j^5)] × [x - 2 - j^4)]

j² = -1 (j ⇒ imaginary number

j^4 = 1

j^5 = j

= [x - 3 + j)] × [x - 2 - 1)]

= [x - 3 + j)] × [x - 3)]

= x² -3x - 3x + 9 + jx -3j

combine like terms

=  x² -6x + jx - 3j + 9
