14. Layla wants to build a wooden box with a volume of 45 cubic centimeters. She started with a width of 3cm, space, c, m and a height of3cm, space, c, m. How long should Layla make the box?


Answer:

The box should be 5 cm long.

Step-by-step explanation:

To get the length of the wooden box that Layla will make, we need to analyze the given data in the problem.

What is asked?

The length of the wooden box.

What are the given?

volume of the wooden box - 45 cm³

width of the box - 3 cm

height of the box - 3 cm

What is the formula?

Volume = l × w × h

To find the length, we are still going to use the formula in finding the volume of a rectangular prism.

l is the length, w is the width and h is the height.

What is the solution?

Substitute the values.

Volume = l × w × h

45 cm³ = l x 3 cm × 3 cm

45 cm³ = l × 9 cm²

45 cm³ / 9 cm² = l × 9 cm² / 9 cm²

5 cm = l

What is the final answer?

The length of the box is 5 cm.

Checking:

To check, substitute all the data.

Volume = l × w × h

45 cm³ = 5 cm x 3 cm × 3 cm

45 cm³ = 45 cm³ ✔

17. Lylia, Layla and Ling collect coins. They exchange coins among themselves according to the following arrangement: Lylia gives Layla as many coins as Layla has and Ling as many coins as Ling has. After that, Layla gives Lylia and Ling as many coins as each of them has, and then Ling give Lylia and Layla as many coins as each has. If each finally has 256 coins, with how many coins does Lylia start?


Answer:

Lylia has 416 coins.

Step-by-step explanation:

Let's do this slowly but surely. Let's represent each person's initial coins as:

a for lylia

b for layla

c for ling

In the first sentence, Lylia gives Layla as many coins as Layla has; and gives Ling as many coins as Ling has. Using our variables, we can represent what happened by:

a - b   → (Lylia gave Layla as many coins as Layla has)

b + b  → (Layla got those coins the same as Layla had)

(a - b) - c → (Lylia gave Ling as many coins as Ling has)

c + c → (Ling got those coins the same as Ling had)

In the end of the first transaction:

Lylia has (a - b - c) coins.Layla has (b+b) = 2b coins.Ling has (c+c) = 2c coins.

On the second sentence, Layla gives Lylia and Ling as many coins as each of them has. We can then represent with our variables what happened:

2b - (a-b-c) → (Layla gave Lylia as many coins as Lylia has)

(a-b-c) + (a-b-c) → (Lylia got those coins the same as Lylia had)

[2b - (a-b-c)] - 2c → (Layla gave Ling as many coins as Ling has)

2c + 2c → Ling got those coins the same as Ling had.

In the end of the second transaction:

Lylia has (a-b-c) + (a-b-c) = (2a - 2b - 2c) coins.Layla has [2b - (a-b-c)] - 2c = 2b - a+b+c - 2c = (3b - a - c) coins.Ling has (2c+2c) = 4c coins.

On the third sentence, Ling gave Lylia and Layla as many coins as each has. We can again represent with our variables what happened:

4c - (2a-2b-2c) → (Ling gave Lylia as many coins as Lylia has)

(2a-2b-2c) + (2a-2b-2c) → (Lylia got those coins the same as Lylia had)

[4c - (2a-2b-2c)] - (3b-a-c) → (Ling gave Layla as many coins as Layla has)

(3b-a-c) + (3b-a-c) → (Layla got those coins the same as Layla has)

In the end of of the third transaction:

Lylia has (2a-2b-2c) + (2a-2b-2c) = (4a - 4b - 4c) coins.Layla has (3b-a-c) + (3b-a-c) = (6b - 2a - 2c) coins.Ling has [4c - (2a-2b-2c)] - (3b-a-c) = (4c - 2a + 2b + 2c - 3b + a + c) = (7c - a - b) coins.

All of this is equal to 256.

We have a system of 3 variables.

4a - 4b - 4c = 256

-2a + 6b - 2c = 256

-a - b + 7c = 256

We can divide the first equation by 4, and the 2nd equation by 2.

a - b - c = 64 → Let's call this equation 1

-a + 3b - c = 128 → Let's call this equation 2

-a - b + 7c = 256 → Let's call this equation 3

Let's add equation 1 to equation 2

[tex]a - b - c + (-a+3b-c) = 64 + 128\\2b - 2c = 192\\b - c = 96[/tex]

b - c = 96 → let's call this equation 4

Let's add equation 1 to equation 3

[tex]a - b - c + (-a-b+7c) = 64 + 256\\-2b + 6c = 320\\ -b + 3c = 160[/tex]

-b + 3c = 160 → Let's call this equation 5

Equation 4 and Equation 5 form a system of 2 variables.

[tex]\left \{ {{b-c=96} \atop {-b+3c=160}} \right.[/tex]

We can add them again to get:

[tex]b - c + (-b + 3c) = 96 + 160 \\2c = 256\\c = 128[/tex]

Ling originally has 128 coins.

Let's substitute c = 128 to equation 4.

[tex]b - c = 96\\\\b - 128 = 96\\b = 224[/tex]

Layla originally has 224 coins.

Let's substitute b = 224, c = 128 to equation 1

[tex]a - b - c = 64\\a - 224 - 128 = 64\\a = 416[/tex]

Lylia originally has 416 coins.

We can work this out again with the problem.

Lylia has 416 coins. Lylia gave Layla 224 coins and Ling 128 coins.

At the end of the first transaction, Lylia has (416-224-128) = 64 coins.

Layla has (224+224) = 448 coins, Ling has (128+128) = 256 coins.

Layla has 448 coins. Layla gave Lylia 64 coins and Ling 256 coins.

At the end of the second transaction, Lylia has (64+64) = 128 coins. Layla has (448-64-256) = 128 coins, Ling has (256+256) = 512 coins.

Ling has 512 coins. Ling gave Layla 128 coins and Lylia 128 coins.

At the end of the third transaction, Layla has (128+128) = 256 coins, Lylia has (128+128) = 256 coins, and Ling has (512-128-128) = 256 coins.

Our solution holds.

Question: Chou is thrice as old as Layla. Three years from now, Chou will be twice as old as Layla. How old is Chou and Layla presently?

A. Chou is 21 years old and Layla is 7 years oldB. Chou is 18 years old and Layla is 6 years oldC. Chou is 15 years old and Layla is 5 years oldD. Chou is 9 years old and Layla is 3 years old

Answer: [tex] \Large \underline {\rm \green{ \: Letter \: D. \: }}[/tex]

Solution: Represent x and y as the ages of Chou and Layla respectively. Make two equations of the given statement.

[tex] \begin{cases}x = 3y& \green{(eq. \: 1)} \\ (x + 3) = 2(y + 3) & \green{(eq. \: 2)}\end{cases}[/tex]

- Substitute x from the first equation to the second equation in terms of y which is the age of Layla.

[tex] \begin{cases}x = 3y\\ (3y + 3) = 2(y + 3)\end{cases}[/tex]

[tex] \begin{cases}x = 3y\\ 3y + 3 = 2(y + 3)\end{cases}[/tex]

[tex] \begin{cases}x = 3y\\ 3y + 3 = 2y + 6\end{cases}[/tex]

[tex] \begin{cases}x = 3y\\ 3y - 2y =6 - 3 \end{cases}[/tex]

[tex] \begin{cases}x = 3y\\y = 3 \end{cases}[/tex]

- Thus, the age of Layla is 3 years old. Then substitute it to the first equation to find Chou's age.

[tex] \begin{cases}x = 3(3)\\y = 3 \end{cases}[/tex]

[tex] \begin{cases}x = 9\\y = 3 \end{cases}[/tex]===============================
