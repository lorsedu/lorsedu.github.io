Let x be Jim's present age

and y be his sister's present age

Formulate equations

x = 2y (Jim is twice as old as his sister.)x + 5 + y + 5 = 91 (Five years from now, the sum of their ages will be 91.)

Substitute x = 2y into x + 5 + y + 5 = 91 and solve.

x + 5 + y + 5 = 912y + 5 + y + 5 = 913y + 10 = 913y = 91 – 103y = 81y = 27Jim's sister's present age is 27.

Jim's present age: (Substitute the value of y into x = 2y)

x = 2yx = 2(27)x = 54Jim's present age is 54.

Checking.

(Jim is twice as old as his sister.)

x = 2y54 = 2(27)54 = 54 ✓

7. £940 is divided between Gavin, Jim & Krutika so that Gavin gets twice as much as Jim, and Jim gets three times as much as Krutika. How much does Jim get?


Answer:

£282

Step-by-step explanation:

Let us represent each amount with the following variables:

g = amount of money Gavin gets

j = amount of money Jim gets

k = amount of money Krutika gets

The expression we have so far is

[tex]940 = g + j + k[/tex]

Gavin gets twice as much as Jim. Using our variables, we have

[tex]g = 2j[/tex]

Jim gets three times as much as Krutika. Using our variables, we have

[tex]j = 3k[/tex]

We can substitute the value of j to the second equation to see how Gavin and Krutika's money are related

[tex]g = 2j\\j = 3k\\\\g = 2(3k)\\g = 6k[/tex]

So, we can say that Gavin gets 6 times as much as Krutika has.

Since we expressed j and g in terms of k, we can substitute those values from the original equation.

[tex]g+k+j = 940\\g = 6k\\j = 3k\\\\6k + k + 3k = 940 \\10k = 940\\k = 94\\\\[/tex]

So Krutika has £94.

To find out how much the Gavin and Jim have, we substitute k =94 to their respective equations.

[tex]k = 94\\\\g = 6k \\g = 6(94)\\g = 564\\\\j = 3k\\j = 3(94)\\j = 282[/tex]

Jim has £282 while Gavin has £564.

29. May is three years younger than Jim. Find their ages if the sum is 81


Problem:

May is three years younger than Jim. Find their ages if the sum is 81.

Solution:

Let x be the Jim's age

Then let x + 3 be the May's age

In representation:

Find their ages if the sum is 81.

(Jim's age) + (May's age) = 81x + (x + 3) = 81

Solve the value of x which is the Jim's age to the problem equations.

x + (x + 3) = 812x + 3 = 812x = 81 - 32x = 78/2x = 39

Substitute the May's age to the present age which is the value of x to the solving the problem equation.

May's age = x + 3 May's age = (39) + 3 May's age = 42

Checking:

Checking the age's if the answer is correct.

(Jim's age) + (May's age) = 81x + (x + 3) = 81(39) + (39 + 3) = 8139 + 42 = 8181 = 81 ✔

Answer:

∴ Therefore, the present age of Jim is 39 yrs old and May is 42 yrs old.
