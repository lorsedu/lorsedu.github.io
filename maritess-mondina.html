27. Maritess is working on the current selling report spreadsheet that includes long text strings. she noticed that the texts do not fit in the cell. what is the correct option in order to fix this issue?


Using wrap text features

In the Microsoft Excel application, If you enter long text in an Excel cell, the text will appear stretched beyond the borders of one or more columns to the right. When you re-enter long text in the cell to the right, the first text appears truncated or covered. The text will appear incomplete and obscure the meaning of the text itself.

You can actually display long text within an Excel cell by using the Wrap text feature. Text will be packed or displayed in multiple lines automatically in an Excel cell.

Here's how to pack long text to stay inside a Microsoft Excel cell: Launch Microsoft Excel and type long text into an Excel cell. Activate the cell containing the text by clicking on it. Make sure you are on the Home tab (1) and then click the Wrap text button (2). Now you can notice that long text spanning multiple columns has been packed onto multiple lines in a cell. If all or part of the packed text is not visible, it is due to insufficient line height. In order for the row height to display all of the packed text, you need to change Excel's row height so that all of the long text is visible.
