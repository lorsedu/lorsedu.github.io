11. Which one of these stores data more data than a DVD? a. CD Rom b. Floppy c. 4gb Flash Drive d. 32gb Memory Card


D. 32gb Memory Card

Storage DevicesDVD

The DVD, which stands for Digital Video Disc or Digital Versatile Disc, is a popular acronym for these digital optical disc data storage formats[8][9]. Despite having the same size as compact discs (CD), DVDs have a far larger storage capacity. A typical DVD can hold up to 4.7 GB of data, whereas variations may hold up to 17.08 GB.

Memory card

A memory card is a type of electronic data storage that commonly uses flash memory to store digital data. These are frequently utilized in portable digital electronics. Instead of utilizing protruding USB flash drives, they enable increasing memory to these devices via a card in a socket. For example, the memory card capacity currently available in the market is 32 GB.

CD Rom

A CD-ROM is a kind of read-only memory made up of a data-filled optical compact disc that has been pre-pressed. CD-ROMs can be read by computers, but not created or deleted. Data (such software or digital video) can only be used on a computer, but some CDs, known as enhanced CDs, can store both computer data and audio, with the latter being playable on a CD player. The CD rom has a capacity of 553-900 MB (12 cm) and 194 MB (8 cm).

Floppy disk

A floppy disk or floppy diskette, more commonly known as a floppy or a diskette, is a type of antiquated disk storage that consists of a thin, flexible disk of a magnetic storage medium inside of a square or nearly square plastic case that is lined with fabric to filter out dust from the spinning disk. The most popular floppy disks are 3.5-inch models with capacities ranging from 800 KB to 2.8 MB.
