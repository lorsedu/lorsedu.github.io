        January 15, 2018, Verification as to completeness of documents.
        January 26, 2018, Orientation of Teachers Applicants.

22. Scheduling. A school has six irregular graduating seniors: Allen (A), Ben (B), Cory (C), Dan (D), Ella (E), and Fe (F). You must prepare a special final exam schedule such that each student can take the exam in all his/her subjects in one day, but students with common subjects cannot be scheduled on the same day. Below are the subjects enrolled by each student: A: English, Science, Politics B: Science, Politics, Philosophy C: Math, Philosophy, Art D: English, French, Art E: Politics, Art, Philosophy F: Math, Science, Music Construct a planar graph and using vertex coloring, find the fewest number of days that the ministration of the special exams can be done. Suggest one possible schedule.


To construct a planar graph, we can first represent each student as a vertex, and connect two vertices with an edge if the corresponding students share at least one subject.

Here is the resulting planar graph:

To find the fewest number of days needed, we need to find the chromatic number of this graph, which is the smallest number of colors needed to color the vertices such that no two adjacent vertices share the same color. In other words, each color represents a different day, and vertices with the same color cannot share a day.

Using a greedy algorithm, we can start with the first vertex and color it with the first color. Then, we move to the next uncolored vertex and color it with the smallest available color that does not conflict with its neighbors. We repeat this process until all vertices are colored.
