4. Lee Jay spent P150.00 on November 1, P170.00 on November 2, P190.00 on November 3, and so on. How much did he spend on November 30?


Answer:

730

Step-by-step explanation:

a1+(n-1)d

150+(30-1)20

150+580

730

Answer:

Lee Jay spent P730,000 on November 30.

Step-by-step explanation:

d = 20,000

A1 = 150,000

n = 30

[tex]A_{n} = A_{1} + ( n - 1) d[/tex]

[tex]A_{30} = 150,000 + ( 30 - 1) 20,000\\ A_{30} = 150,000 + (29) 20,000\\ A_{30} = 150,000 + 580,000\\ A_{30} = 730,000[/tex]
