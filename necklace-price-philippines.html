1. A necklace was sold for Php3150.00 with a 40% discount.what was the original Price of the necklace?​


ANSWER


Php5250.00

SOLUTION

To determine the original price of the necklace, you can use the following formula:

Original price = selling price / (1 - discount%)

Plugging in the given values, you get:

Original price = 3150 / (1 - 40%) = 3150 / (1 - 0.4) = 3150 / 0.6 = 5250

Therefore, the original price of the necklace was Php5250.00.

C= Cost or original price

M= Mark-up or increase

MR= Mark-up rate or percentage increase

SP= Selling price

To find the new selling price after a 125% increase in price:

1. Ring:

C = ₱100,000.00

MR = 125% = 1.25

M = C x MR = ₱100,000.00 x 1.25 = ₱125,000.00

SP = C + M = ₱100,000.00 + ₱125,000.00 = ₱225,000.00

The selling price of the ring after a 125% increase is ₱225,000.00.

2. Earrings:

C = ₱15,000.00

MR = 125% = 1.25

M = C x MR = ₱15,000.00 x 1.25 = ₱18,750.00

SP = C + M = ₱15,000.00 + ₱18,750.00 = ₱33,750.00

The selling price of the earrings after a 125% increase is ₱33,750.00.

3. Necklace:

C = ₱23,000.00

MR = 125% = 1.25

M = C x MR = ₱23,000.00 x 1.25 = ₱28,750.00

SP = C + M = ₱23,000.00 + ₱28,750.00 = ₱51,750.00

The selling price of the necklace after a 125% increase is ₱51,750.00.

4. Bracelet:

28. Edna already owns 4 necklaces, and additional necklaces are priced at 2 for a dollar. Is the total number of necklaces Edna owns proportional to the amount of money Edna spends on additional necklaces?


Answer:

Yes.

Explanation:

Two terms are proportional if one term increases with the other, or as one term increases, the other decreases.

Using this basis, we know that the number of necklaces is proportional to the amount of money spent.

As the amount increases by a dollar, the necklaces increase by 2.

We can also determine if this is proportional if we can come up with a linear equation that relates the two terms.

Let x = necklaces; let y = amount spent for additional necklaces.

Using what I previously said in the third paragraph, y = x/2. But, we need to add 4 to this since there are 4 necklaces already to start with. Thus, y = x/2 + 4.

To summarize, yes; the number of necklaces is proportional to the amount spent on additional necklaces.
