28. Gawain 7. Pagkompyut sa Buwis sa Kita 1. Si G. John Christian Espiritu ay may buwanang kita na Php. 29,800.00. Ang mga sumusunod ay ang kanyang binabayarang kontribusyon: SSS - Php. 2,600.00, PhilHealth Php. 100, Pag-ibig Fund Php. 100. Siya ay may asawa na hindi nagtatrabaho at may 3 anak na may edad na 3, 6, 9 na taong gulang.a. Magkano ang kabuuang kontribusyon no G. Espiritu?b. Magkano ang kanyang kabuuang eksemsyon?c. Magkano ang kanyang kabuuang taxable income? (Gross Taxable Income)d. Magkano ang kabuuang buwis na kanyang babayaran?e. Magkano ang kanyang buwanang withholding tax?​


WITHHOLDING TAX

Mayroong buwanang kita si G.John Christian Espiritu na nagkakahalaga ng Php 29,800. Mayroon siyang  kabuuang kontribusyon  nag nagkakahalaga ng Php 4,055.50 para sa SSS, PhilHealth, at Pag-IBIG. Mayroon siyang Php 125,000 na kabuuang eksemsyon. Dahil mas mataas ang halaga ng eksemsyon at kontribusyon kaysa sa kanyang gross monthly income, hindi siya magbabayad ng buwis at withholding tax.

KABUUANG KONTRIBUSYON

SSS Kontribusyon = 12% x Php 22,000.00 = Php 2,640.00

PhilHealth Kontribusyon =  = 2.75% x Php 29,800.00 = Php 819.50

Pag-IBIG Kontribusyon = = 2% x Php 29,800.00 = Php 596.00

Kabuuang Kontribusyon = Php 2,640.00 + Php 819.50 + Php 596.00 = Php 4,055.50

KABUUANG EKSEMSYON

Personal Exemption = Php 50,000.00

Additional Exemption = Php 25,000.00 x 3 = Php 75,000.00

Kabuuang Eksemsyon = Php 50,000.00 + Php 75,000.00 = Php 125,000.00

KABUUANG TAXABLE INCOME

Kabuuang Taxable Income:

Gross Monthly Income = Php 29,800.00

Total Contributions = Php 4,055.50

Taxable Income = Gross Monthly Income - Total Contributions - Personal Exemption - Additional Exemption

Taxable Income = Php 29,800.00 - Php 4,055.50 - Php 50,000.00 - Php 75,000.00

Kabuuang Taxable Income = -Php 99,255.50 (negative dahil ang mga eksemsyon at kontribusyon ay mas mataas kaysa sa gross monthly income)

KABUUANG BUWIS NA BABAYARAN

Dahil negatibo ang taxable income, wala siyang babayarang buwis.

BUWANANG WITHHOLDING TAX

Dahil wala siyang babayarang buwis, wala rin siyang buwanang withholding tax.
